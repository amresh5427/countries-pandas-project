# 🌍 Countries Data Analysis

A data analysis project built with Python, exploring global country data
using Pandas and NumPy. The dataset covers 190+ countries with 64 attributes
including population, regions, democracy scores, political leaders, and currencies.

## 📁 Project Structure

| File | Description |
|------|-------------|
| `country.ipynb` | Jupyter Notebook with full analysis |
| `Countries.csv` | Dataset with 190+ countries |

## 📊 Analysis Performed

- 🏙️ Country with the highest and lowest population
- 🏛️ Capital cities of most/least populated countries
- 📈 Top 5 countries ranked by democracy score
- 🌐 Total number of regions and country count per region
- 👤 Political leaders of the most populated countries
- ❓ Countries with unknown political leaders
- 🗺️ Countries within specific regions (e.g. Eastern Europe)

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Jupyter Notebook

## 📂 Dataset Features

The dataset includes 64 attributes such as:
`country`, `population`, `capital_city`, `region`,
`democracy_score`, `political_leader`, `currency`, and more.
